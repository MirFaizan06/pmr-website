Why Google Drive?

Patient Records Manager is a personal project built to help real clinics — not a commercial product. Using Google Drive keeps the download completely free, avoids server costs, and means the file is available as long as Google is. No ads, no tracking, no account required to download.

What's inside

Everything a busy clinic needs, without the complexity of enterprise software.

Patient Records

Store and retrieve complete patient histories — visits, diagnoses, medications — instantly searchable by name, phone, or ID.

Password Protected

All records are secured behind a password. Only authorised users can access the app or use the CLI tool.

Face Unlock

Log in with facial recognition — fully local, no cloud, no data sent anywhere. Falls back to password if camera is unavailable.

Prescription Printing

Generate formatted prescriptions on A4 paper with your clinic's letterhead, patient details, and investigation checklist.

CLI Tool (pmr)

A standalone command-line interface included in the installer. Search patients, run backups, and query records from any terminal.

Works Offline

All data is stored locally. No cloud sync, no subscriptions. The app works fully without an internet connection.
